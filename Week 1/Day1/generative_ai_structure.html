Generative AI Project Structure

config/
model_config.yaml – LLM providers, models, & parameters.
logging_config.yaml – Logging setup and levels.
data/
cache/ – Cached responses and intermediates.
embeddings/ – Generated vector embeddings.
vectordb/ – Vector database indexes (FAISS, ChromaDB).
src/
LLM abstraction and integrations.
base_llm.py – Common LLM interface.
gpt_client.py – OpenAI GPT client.
claude_client.py – Anthropic Claude client.
local_llm.py – Local models (Ollama, vLLM).
model_factory.py – Model selection and instantiation.
prompts/
templates.py – Reusable prompt templates.
chain.py – Multi-step prompt chaining.
rag/
Retrieval-Augmented Generation components.
embedder.py – Embedding generation.
generate.py – Document indexer.
vector_store.py – Vector DB retrieval.
indexer.py – Relevance scoring and metadata.
processing/
chunking.py – Text splitting.
cleaner.py – Text extraction and cleaning.
preprocessor.py – Data normalization.
postprocessor.py – Output formatting.
inference/
inference.py – Inference engine and environment setup.
docs/
README.md – Overview and usage.
SETUP.md – Installation and setup.
scripts/
setup_env.sh – Environment setup.
run_tests.sh – Test execution.
build_embeddings.py – Embedding generation.
cleanup.py – Cleanup tasks.
Root Files
.gitignore – Git exclusions.
Dockerfile – Container configuration.
docker-compose.yml – Multi-service setup.
requirements.txt – Python dependencies.
