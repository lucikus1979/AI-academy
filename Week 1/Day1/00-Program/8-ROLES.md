## Role Selection

### How to Choose

1. **Day 4:** You'll get detailed previews of each role
2. **Consider:** Your background, interests, and career goals
3. **Discuss:** With your mentor if unsure
4. **Decide:** By end of Day 4

### Selection Criteria

| If You... | Consider |
|-----------|----------|
| Love building end-to-end | FDE |
| Obsess over code quality | AI-SE |
| Think strategically | AI-PM |
| Think like a hacker | AI-SEC |
| Love beautiful interfaces | AI-FE |
| Advocate for users | AI-DX |
| Love experiments & metrics | AI-DS |
| Turn data into stories | AI-DA |
