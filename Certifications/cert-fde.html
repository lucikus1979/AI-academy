💡 Study Tips for FDE Certification
1
Start with AI-900 for Quick Win
AI-900 is beginner-level and can be passed in 2-4 weeks with focused study on Microsoft Learn. Build confidence and get a certification under your belt before tackling the more demanding AWS Developer Associate exam.
2
Build Real Projects on AWS
Deploy actual applications using Lambda, API Gateway, DynamoDB. Hands-on experience matters more than memorizing services. Build a REST API, connect it to a database, and deploy it end-to-end to truly understand the platform.
3
Use Free Tier Strategically
AWS Free Tier gives 12 months of core services. Practice building and deploying real apps without cost. Use Lambda (1M free requests/month), DynamoDB (25GB free), and S3 (5GB free) to build portfolio projects that also prepare you for the exam.
📅 4-Week Quick Start Study Plan
Week 1 — AI-900 Foundations
Azure AI Concepts & Fundamentals
  • Study Azure AI concepts on Microsoft Learn AI-900 learning path
  • Understand AI workloads and machine learning principles
  • Explore Computer Vision, NLP basics, and conversational AI
  • Complete hands-on labs in Microsoft Learn sandbox environment
Week 2 — AI-900 Exam Prep
Practice Tests & Pass AI-900
  • Take AI-900 practice tests on Microsoft Learn
  • Review weak areas and revisit relevant modules
  • Schedule and pass the AI-900 exam
  • Set up an AWS Free Tier account and explore the console
Week 3 — AWS Developer Deep Dive
Core AWS Services & Sample Apps
  • Study core AWS services: Lambda, API Gateway, DynamoDB, S3, IAM
  • Start building sample applications on AWS Free Tier
  • Deploy a REST API using Lambda + API Gateway
  • Read the AWS Developer Associate exam guide and identify key domains
Week 4 — AWS Developer Practice
Mock Exams, Portfolio & CI/CD
  • Complete full mock exams on AWS Skill Builder
  • Deploy a portfolio project end-to-end (frontend + API + database)
  • Review CI/CD concepts: CodePipeline, CodeBuild, CodeDeploy
  • Schedule your AWS Developer Associate exam
