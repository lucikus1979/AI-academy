Study Tips

1
Master Microsoft Sentinel First
SC-200 centers on Microsoft Sentinel, Defender XDR, and security operations. Hands-on lab time is essential.
2
CompTIA AI+ Covers Agentic Security
40% of the exam covers agentic AI content, making it directly relevant to securing AI systems.
3
Connect Security to AI Governance
Understanding both SC-200 (SOC operations) and AI+ (AI risks) positions you uniquely for AI security consulting.
📅

4-Week Study Plan

Week 1 — SC-200 Foundations
Microsoft Sentinel workspace, KQL queries, threat detection basics
  • Set up Microsoft Sentinel workspace and connect data sources
  • Learn KQL (Kusto Query Language) fundamentals for log analysis
  • Study threat detection basics: analytics rules, incidents, and alerts
  • Complete Microsoft Learn SC-200 learning path (Modules 1-2)
Week 2 — SC-200 Deep Dive
Incident management, threat hunting, Defender XDR integration
  • Practice incident management workflows in Microsoft Sentinel
  • Learn threat hunting techniques using KQL and built-in hunting queries
  • Study Microsoft Defender XDR integration with Sentinel
  • Complete Microsoft Learn SC-200 learning path (Modules 3-5)
Week 3 — SC-200 Exam + AI+ Start
Practice exams for SC-200, begin CompTIA AI+ study
  • Take SC-200 practice exams and review weak areas
  • Schedule and sit the SC-200 certification exam
  • Begin CompTIA AI+ study: AI concepts, ethics, and security risks
  • Review OWASP LLM Top 10 as foundation for AI+ security content
Week 4 — CompTIA AI+
Agentic AI security, responsible AI, AI governance, mock exams
  • Study agentic AI security: autonomous agent risks and safeguards
  • Learn responsible AI principles and AI governance frameworks
  • Complete CompTIA CertMaster Practice sessions and mock exams
  • Schedule and sit the CompTIA AI+ certification exam
